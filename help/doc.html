Функции получения данных

getDepth()

Функция получения данных с датчика глубина. Значения глубина возвращается в см. Тип возвращаемого значения - float. Датчик давления установлен внутри модуля бортового компьютера.

getYaw()

Функция получения данных о магнитном курсе. Значени
я курса изменяются от -180 до 180 градусов. Тип возвращаемого значения – float. Компас установлен внутри модуля бортового компьютера.

getAS()

Функция получения данных об угловой скорости по оси X. Тип возвращаемого значения - float. AS - Angular speed. Угловая скорость может использоваться для реализации дифференциальной составляющий PD-регулятора для стабилизации по курсу.

getPitch()

Функция получения данных о дифференте. Значения дифферента изменяются от -180 до 180 градусов. Тип возвращаемого значения - float. Дифферент соответствует продольному отклонению модуля бортового компьютера.

getRoll()

Функция получения данных о крене. Значения курса изменяются от -180 до 180 градусов. Тип возвращаемого значения - float. Дифферент соответствует поперечному отклонению модуля бортового компьютера.

showImageOne(Camera &camera) / showImageOne(cv::Mat &image)

Показывает изображение с камеры в первом окне, если компиляция и запуск осуществляются на компьютере.

showImageTwo(Camera &camera) / showImageTwo(cv::Mat &image)

Показывает изображение с камеры во втором окне, если компиляция и запуск осуществляются на компьютере.

showImageThree(Camera &camera) / showImageThree(cv::Mat &image)

Показывает изображение с камеры в третьем окне, если компиляция и запуск осуществляются на компьютере.

Функции инициализации

setupThrusters(int idOne, int idTwo, int idThree, int idFour)

Функция инициализации движителей. Аргументы функции – номера движителей. Каждый движитель в наборе имеет свой номер 10, 20, 30 или 40 (для дополнительного движителя). Соответственно в качестве аргументов используются значения 10, 20, 30 или 40. Инициализация производится один раз перед началом работы с движителями.

setThrusterPower(THRUSTER thruster, int power)

Функция установки тяги на движитель. Аргументы функции - движитель и тяга. Значения тяги можно установить в промежутке от -100 до 100.

applyThrusterPower()

Функция применения тяги на движители. Устанавливает тягу на движители, заданную при помощи функции setThrusterPower.

Функции распознавания

detectLine(Camera &camera)

Функция распознавания линии. Тип возвращаемого значения – структура Object. Аргументы - Camera.

detectTriangle(Camera &camera)

Функция распознавания треугольника. Тип возвращаемого значения – структура Object. Аргументы – Camera.

detectRectangle(Camera &camera)

Функция распознавания четырёхугольника. Тип возвращаемого значения – структура Object. Аргументы – Camera.

detectCircle(Camera &camera)

Функция распознавания окружности. Тип возвращаемого значения – структура Object. Аргументы – Camera.

detectMotion(Camera &camera)

Функция распознавания смещения кадров. Тип возврата – структура Motion. Аргументы – Camera.

detectObject(Camera &camera)

Функция распознавания объекта на кадре (треугольник, четырехугольник, круг). Тип возвращаемого значения – структура Object. Аргументы – Camera.
